## torch.linspace(n1, n2, n)

 是 PyTorch 中的一个张量操作函数，用于生成一维均匀间隔的数值序列。将区间[n1, n2]等分成n的tensor

```python
k = torch.linspace(0, 4, 9)
k
# tensor([[[1., 1., 1.],
#         [1., 1., 1.]]])
# tensor([[0., 0., 1., 1.],
#        [0., 0., 1., 1.]])
# tensor([0.0000, 0.5000, 1.0000, 1.5000, 2.0000, 2.5000, 3.0000, 3.5000, 4.0000]) 
```
